Manual Testing Instructions

1. Theme Toggle Test
  1. Open http://localhost:56349
  2. Click the theme toggle button in the header
  3. Verify smooth animation with spring physics (bouncy effect)
  4. Verify indicator rotates 180 degrees
  5. Verify all colors transition smoothly
  6. Reload the page and verify theme persists
2. Video Selection Test
  1. Click "Select Video" button
  2. Choose a video file (or drag and drop onto the drop zone)
  3. Verify file info displays correctly with monospace font
  4. Verify all existing compression features still work
3. Responsive Design Test
  1. Open browser dev tools (F12)
  2. Toggle device toolbar (Ctrl/Cmd + Shift + M)
  3. Test various screen sizes (mobile, tablet, desktop)
  4. Verify layout adapts correctly
  5. Verify preset buttons stack on mobile
4. Interaction Test
  1. Hover over buttons - verify lift effect (translateY(-2px))
  2. Hover over drop zone - verify border color changes
  3. Click buttons - verify scale(0.98) active state
  4. Test keyboard navigation - verify focus-visible styles
